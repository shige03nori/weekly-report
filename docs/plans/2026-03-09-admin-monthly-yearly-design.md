# 管理者 月次・年次提出状況ビュー 設計

## 概要
管理者が月単位・年単位で全ユーザーの週報提出状況をマトリクスで把握できる画面を追加する。

## 新規画面

### 月次ビュー `/mgmt/monthly/`
- 年・月セレクタで対象月を選択
- 行: 一般ユーザー一覧
- 列: その月に含まれる週（4〜5列）
- セル: ✓（提出済み・緑）/ ✗（未提出・赤/グレー）
- 末尾に合計列（例: 3/4）
- モバイル: table-responsive 対応

### 年次ビュー `/mgmt/yearly/`
- 年セレクタで対象年を選択
- 行: 一般ユーザー一覧
- 列: その年の全週（〜52週）、月ごとにグループ化
- セル: ✓/✗
- 末尾に年間合計列
- 横スクロール対応

## 管理者ナビゲーション
全管理者ページに共通ナビを追加（`admin_base.html` として切り出し）:
- 提出状況（週次）
- 月次ビュー（新規）
- 年次ビュー（新規）
- 集計
- 質問管理
- ユーザー管理

## 実装ファイル
- `reports/views_admin.py` — 2つのビュー追加
- `reports/urls.py` — 2つのURL追加
- `templates/reports/admin_base.html` — 管理者共通ナビ（新規）
- `templates/reports/admin_monthly.html` — 月次テンプレート（新規）
- `templates/reports/admin_yearly.html` — 年次テンプレート（新規）
- 既存管理者テンプレート — `admin_base.html` を継承するよう変更
